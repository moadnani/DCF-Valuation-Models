# Apple Inc. DCF Valuation

## Overview

This project is a five-year discounted cash flow (DCF) valuation model for Apple Inc.

The model forecasts operating performance using revenue growth, operating margin and reinvestment assumptions before estimating enterprise value using the Free Cash Flow to Firm (FCFF) approach.

The objective of this project was to demonstrate practical financial modelling skills using publicly available financial statements.

---

## Valuation Methodology

- Historical financial statements (FY2021–FY2025)
- Five-year operating forecast
- FCFF valuation
- WACC estimated using CAPM
- Gordon Growth terminal value
- Sensitivity analysis across WACC and terminal growth

---

## Key Assumptions

| Assumption | Value |
|------------|------:|
| Forecast Period | 5 Years |
| Terminal Growth | 4% |
| Risk-Free Rate | 4.3% |
| Equity Risk Premium | 4.23% |
| Beta | 1.1 |
